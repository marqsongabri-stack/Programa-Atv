# 🛒 Lista de Compras Inteligente

Sistema web para organização de compras com categorias, filtros e status.

## 🚀 Funcionalidades
- Adicionar itens
- Remover itens
- Marcar como comprado
- Filtrar por categoria
- Filtrar por status

## 🏗 Arquitetura
Frontend (HTML/CSS/JS) + Backend (Node.js + Express) + SQLite

## ▶️ Como rodar

### Backend
npm install  
node server.js  

### Frontend
Abrir index.html

## 🔌 API
GET /api/items  
POST /api/items  
DELETE /api/items/:id  
PATCH /api/items/:id/toggle  

## 🧠 Categorias
- Alimentos
- Hortifruti
- Bebidas
- Limpeza
- Higiene
- Padaria
